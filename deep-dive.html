API Map

Odin API

שער זהויות ואימות עם מודולי auth, events, onboarding ו-magic-link.

  • POST /auth/login
  • POST /webhooks/odin
  • POST /magic-link/send

RideOS API

ניהול נסיעות, נהגים ונוסעים, מעקב ומערך דירוגים.

  • POST /rides/create
  • POST /rides/assign
  • GET /tracking/live

LocalBiz API

אוטומציית הקמת אתרים לעסקים כולל יצירת תוכן וחיבור סליקה.

  • POST /sites/generate
  • POST /ai/content
  • POST /payments/link

Sync Dispatch API

ניהול הקצאות בזמן אמת עם אירועי נהגים והתראות.

  • POST /dispatch/event
  • POST /drivers/notify

מודל אבטחה

JWT Federation

Odin מנפיק RS256 וכל השירותים מאמתים חתימה עם מפתח ציבורי.

  • Short TTL Access Tokens
  • Scope-based Authorization

Service-to-Service Keys

קריאות פנימיות מוגנות ב-X-Internal-Key ו-X-Vault-Internal-Key.

  • Key Rotation
  • IP Allow Lists

Webhook Hardening

אימות חתימה, מניעת replay וחסימת payload חריג.

  • HMAC Verification
  • Nonce + Timestamp

Data & Migrations

PostgreSQL Domains

טבלאות זהות, KYC, ארנקים ותנועות מחולקות לפי שירותים.

  • Alembic revision history
  • Backward-compatible schema changes

Redis Strategy

ניהול סשנים, cache חם ו-rate-limit לפי endpoint.

  • Session TTL Policies
  • Key namespace isolation

Mongo Event Log

שכבת אירועים ל-dispatch, audit וניתוח SLA.

  • Assignment snapshots
  • Outbox pattern

תפעול ואמינות

Retries

Worker ייעודי ל-retry על יצירת ארנקים וקריאות חוצות שירותים.

  • Exponential backoff
  • Dead-letter capture

Observability

לוגים מובנים, trace-id, ומדדי latency/error לכל API.

  • Request correlation
  • Webhook failure dashboards

Messaging Fallback

אם Meta נכשל, המערכת עוברת אוטומטית ל-Twilio.

  • Health probe routing
  • Fallback alerting
